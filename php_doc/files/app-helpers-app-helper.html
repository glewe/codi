Function to log data to the JavaScript console.

consoleLog(string $module, string $function, string $variableName, mixed $content[, bool $with_script_tags = true ]) : void

This function generates a JavaScript console.log statement with the provided module name, variable name, and content. The generated JavaScript code can optionally be wrapped in script tags.

Parameters
$module : string

The name of the module where the log is coming from.

$function : string

The name of the function where the log is coming from.

$variableName : string

The name of the variable to be logged.

$content : mixed

The content to be logged. This can be any data type that can be JSON encoded.

$with_script_tags : bool = true

Optional. Whether to wrap the JavaScript code in script tags. Default is true.

Return values
void —

This function does not return a value. It echoes the generated JavaScript code.

Parameters

The route for which the help page URL is needed.

Return values

The URL of the help page. If a specific help page URL is set for the given URI in the 'HelpPages' config, that URL is returned. Otherwise, the default help page URL from the 'AppInfo' config is returned.
