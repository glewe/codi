------------------------------------------------------------------------ Check license switch.

public bool $checkLicense = false

This switch enables or disables the license check. If set to true, the application will check the license key with the license server.

If set to false, the application will not check the license key even though you might have added it as a filter for a route. Also, the license page will not be displayed.
